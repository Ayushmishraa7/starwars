## General info

App allows you to search characters from a Star Wars, add/remove them to/from your favorites list.

## Build using:

- HTML
- CSS
- React
- Redux
- [Star Wars Api](https://swapi.dev/)
- [React-Tostify](https://fkhadra.github.io/react-toastify/how-to-style)
